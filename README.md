## 简介

**杉泽 AI 提示库 V2.0** 是一款专为零基础创作者设计的 AI 漫剧提示词生成工具。

- 不懂英文 · 不懂技术 · 选感觉就能出电影级画面
- 提示词以中文为主，专业影视术语保留英文并附说明
- 内置 30 年导演经验提炼的构图打光建议

**支持平台：** Midjourney · Stable Diffusion · Runway · Sora · ComfyUI · NovelAI · 即梦 AI · 可灵

---

## 功能

- **12 种内容类型** — 热血动漫、爱情漫画、悬疑惊悚、奇幻史诗、科幻赛博、恐怖黑暗、日常治愈、古装武侠、都市偶像、机甲战争、仙侠玄幻
- **12 种情绪感觉** — 每种都有"朦胧感觉"描述，用直觉选即可
- **场景细节** — 性别、气质、场景、天气、镜头、色调、画面比例
- **中英双语输出** — 中文提示词供理解，英文提示词用于生成精准画面
- **导演笔记** — 12 套专业构图打光经验
- **一键导出** — 保存完整 .txt 文件

2. 进入仓库页面 → **Add file → Upload files**
3. 将解压后的全部文件拖入上传区域
4. 点击 **Commit changes**
5. 开启 GitHub Pages：Settings → Pages → Branch: main → Save
